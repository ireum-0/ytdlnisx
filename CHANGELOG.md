# YTDLnisX Changelog

> # 1.8.8 (2026-02)

### 주요 변경사항
- 히스토리 검색/필터 시스템 확장
    - 검색 입력 시트 분리(`title / keywords / creator`)
    - 검색 필드별(제목/키워드/크리에이터) 토큰 검색 정교화
    - 유튜버/키워드/그룹 모드와 검색 조건 동시 적용 로직 개선
    - 플레이어 진입 시 현재 히스토리 컨텍스트(검색/필터) 전달 및 하단 목록 반영 보강

- 유튜버 그룹 계층 구조 도입
    - 유튜버 그룹 부모-자식 관계 데이터 모델 추가
    - 그룹 트리(루트/하위) 표시 및 하위 그룹 단계 표기(예: `ㄴ`, `ㄴㄴ`) 지원
    - 그룹 부모 편집, 다중 그룹을 상위 그룹에 추가 기능 지원
    - DB 마이그레이션 v41 적용 및 스키마 갱신

- 키워드/유튜버 목록 탐색 및 노출 제어 강화
    - 첫 목록 기본 노출 규칙 정비(루트/무부모 우선)
    - 하위 항목 "첫 목록에 표시/해제" 지원 및 영구 저장
    - 키워드/유튜버 롱프레스 기반 하위 항목 선택/제외 동작 개선
    - 뒤로가기 복원 안정화(진입 출처 기준 상태 복원)

- 선택/액션 모드 동작 개선
    - 그룹/비그룹 혼합 선택 시 선택 수 및 실제 액션 처리 정합성 개선
    - 그룹 추가/제거 시 그룹 멤버 확장 반영
    - 관련 컨텍스트 메뉴 항목 추가(첫 목록 표시/숨김, 부모 그룹 편집 등)

- 플레이어(PiP/백그라운드) 안정성 개선
    - 홈 버튼 전환 시 PiP 진입 신뢰성 개선(초기 파라미터 선적용/폴백)
    - `resizeableActivity` 명시 및 PiP 파라미터 갱신 로직 보강
    - PiP 실패 시 백그라운드 재생 폴백 유지
    - 재생 정보/알림 동기화 및 큐 컨텍스트 필터링 보정

- 백업/복원 확장
    - 유튜버 그룹 관계(`youtuber_group_relations`) 백업/복원 지원
    - 그룹/멤버/관계 복원 시 ID 매핑 및 무결성 처리 보강
    - 예약 다운로드(`scheduled`) 복원 누락 수정 및 복원 후 스케줄 워커 재기동
    - 병합 복원 시 설정 전체 초기화되지 않도록 동작 보정(reset에서만 clear)
    - `Set<String>` 설정값(JSON 배열) 백업/복원으로 쉼표 포함 값 손상 방지
    - 백업 포맷 버전(`backup_format_version`) 추가 및 복원 파서 처리 보강

### 참고
- 내부 구조 변경 폭이 커 히스토리/그룹/검색 화면 중심으로 동작 점검을 권장합니다.


> # 1.8.7.5 (2026-02)

### 주요 변경사항
- 히스토리 로컬 분류 개편
    - 재생목록(Playlist) 기능 완전 제거
    - 키워드 중심 분류/검색 구조로 통합
    - 키워드 길게 눌러 다중 선택 후 삭제 지원

- 키워드 분류/탐색 강화
    - 키워드 카드를 재생목록 카드 수준으로 개선 (썸네일/정렬 보정)
    - 키워드 계층 표시 규칙 개선 (선택한 단계 기준으로 하위 단계 점진 표시)
    - 유튜버/아티스트 메타데이터와 키워드 연계 인식 보정
    - 키워드 그룹 화면에도 동일 규칙 적용

- 재생 안정성 개선
    - 백그라운드 재생 유지 로직 보강
    - PiP 전환/확장 안정성 개선
    - PiP 및 알림의 현재 재생 정보 동기화 보완

- 백업/복원 정리
    - 키워드 그룹/멤버, 유튜버 그룹/멤버/메타 백업·복원 반영
    - 재생목록 관련 백업 데이터 경로 정리

- 앱 브랜딩 및 아이콘 업데이트
    - 앱 이름 표기 `YTDLnisX`로 통일
    - 런처/앱 내 로고 및 아이콘 디자인 개선


> # 1.8.7.4 (2026-02)

### 주요 변경사항
- 권한 안내 플로우 개선
    - 파일 권한 안내에서 확인 시 설정 화면으로 이동하도록 개선
    - 다운로드 기본 경로 설정 흐름 개선

- 앱 데이터 백업/복원 확장
    - 커스텀 썸네일 파일 백업/복원 개선
    - downloads 백업 시 custom thumbnails 포함 및 복원 경로 매핑 반영

- 히스토리 탭 전환 스크롤 안정화
    - 유튜버 필터 진입/복귀 시 스크롤 위치 안정성 보강
    - 최초 진입 직후 상단 고정 보강

### 참고
- 커스텀 썸네일 백업은 별도 체크 항목이 아니며 downloads 항목에 포함됩니다.
- 이전 버전 백업 파일도 기존 방식으로 복원 가능합니다.
